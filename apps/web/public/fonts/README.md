# Neue Montreal Font Files

Place your Neue Montreal font files here:

## Required files:

- NeueMontreal-Regular.woff2
- NeueMontreal-Italic.woff2
- NeueMontreal-Medium.woff2
- NeueMontreal-Bold.woff2

## Where to get the fonts:

Neue Montreal is available from Pangram Pangram:
https://pangrampangram.com/products/neue-montreal

## Alternative (free):

If you don't have Neue Montreal, you can use Inter as a free alternative.
See `apps/web/FONTS_README.md` for instructions.
